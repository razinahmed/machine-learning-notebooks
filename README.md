## 📈 Contents

1. **`01_Exploratory_Data_Analysis.ipynb`**
   - Covers Pandas profiling, Seaborn visualizations, and dealing with missing variables.
2. **`02_Random_Forest_Classifier.ipynb`**
   - A robust pipeline for training, cross-validating, and tuning a Random Forest model on tabular data.
3. **`03_PyTorch_CNN_MNIST.ipynb`**
   - A ground-up Convolutional Neural Network built in PyTorch to classify the MNIST handwritten digit dataset.

## 🚀 Usage

To run these notebooks locally, set up a Conda or Python virtual environment:

```bash
git clone https://github.com/razinahmed/machine-learning-notebooks.git
cd machine-learning-notebooks
pip install -r requirements.txt
jupyter notebook
```

*Don't have a local GPU? You can open any of these notebooks directly in Google Colab!*
